# تور تورک | Tour Torque

![اسکرین‌شات از وبسایت](./public/images/Screenshot.png)


**تور تورک** یک وب‌سایت گردشگری مدرن، واکنش‌گرا و کاربرپسند است که با استفاده از **React** و **Tailwind CSS** طراحی و پیاده‌سازی شده. هدف این پروژه نمایش تورهای مسافرتی، جزئیات سفرها و فراهم کردن امکان رزرو آسان برای کاربران می‌باشد.

## 🚀 نسخه‌ی آنلاین

[مشاهده‌ی دموی زنده](https://tour-torque-reactts.vercel.app/)  

✨ امکانات و ویژگی‌ها
طراحی واکنش‌گرا: کاملاً مناسب برای موبایل، تبلت و دسکتاپ.
نمایش تورها: لیست تورها همراه با تصاویر و توضیحات جامع.
جزئیات تور: صفحه‌ی اختصاصی برای هر تور شامل مدت سفر، قیمت و برنامه‌ی سفر.
جستجو و فیلتر: امکان یافتن تورها بر اساس مقصد، تاریخ و قیمت.
رابط کاربری: روان، زیبا و همراه با انیمیشن‌های ملایم.
کامپوننت‌های قابل استفاده مجدد: برای توسعه‌ی سریع‌تر و نگهداری بهتر.
بهینه‌سازی شده: عملکرد سریع و تجربه‌ی کاربری عالی.
🛠️ تکنولوژی‌های استفاده شده
React: کتابخانه‌ی جاوااسکریپت برای ساخت رابط کاربری.
Tailwind CSS: فریم‌ورک کاربردی CSS برای طراحی سریع.
React Router: مدیریت مسیرها و ناوبری در برنامه.
Axios: (در صورت استفاده) برای درخواست‌های HTTP.
React Icons: (در صورت استفاده) برای آیکون‌های متنوع.
سایر کتابخانه‌ها: (لیست کتابخانه‌های استفاده شده را اضافه کنید)
📦 نصب و راه‌اندازی
برای اجرای پروژه روی سیستم خود، مراحل زیر را دنبال کنید:

پیش‌نیازها: اطمینان حاصل کنید که Node.js و npm روی سیستم شما نصب است.
دریافت کد: یک کپی از مخزن را clone کنید:
bash
    git clone https://github.com/MobinDe/tour-torque.git
ورود به پوشه پروژه:
bash
    cd tour-torque
نصب وابستگی‌ها:
bash
    npm install
اجرای پروژه در حالت توسعه:
bash
    npm start
پس از اجرای این دستور، پروژه در آدرس http://localhost:3000 در دسترس خواهد بود.

📁 ساخت نسخه‌ی نهایی (Production)
برای ساخت فایل‌های بهینه‌شده جهت بارگذاری روی سرور، از دستور زیر استفاده کنید:

bash
npm run build
فایل‌های نهایی در پوشه‌ی build/ قرار می‌گیرند و آماده‌ی بارگذاری روی هر سرویس میزبانی استاتیک (مانند Vercel، Netlify) هستند.
