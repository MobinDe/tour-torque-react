# تور تورک | Tour Torque

![اسکرین‌شات پروژه](./screenshot.png)

**تور تورک** یک وب‌سایت گردشگری مدرن، واکنش‌گرا و کاربرپسند است که با استفاده از **React** و **Tailwind CSS** طراحی و پیاده‌سازی شده. هدف این پروژه نمایش تورهای مسافرتی، جزئیات سفرها و فراهم کردن امکان رزرو آسان برای کاربران می‌باشد.

## 🚀 نسخه‌ی آنلاین

[مشاهده‌ی دموی زنده](https://tour-torque-reactts.vercel.app/)  


## ✨ امکانات و ویژگی‌ها

- طراحی کاملاً واکنش‌گرا (ریسپانسیو) مناسب موبایل، تبلت و دسکتاپ
- نمایش لیست تورها با تصاویر و توضیحات
- صفحه‌ی جزئیات تور (مدت سفر، قیمت، برنامه‌ی سفر)
- فرم جستجو و فیلتر تورها بر اساس مقصد، تاریخ و قیمت
- رابط کاربری روان و زیبا با انیمیشن‌های ملایم
- استفاده از کامپوننت‌های قابل استفاده‌ی مجدد
- بهینه‌سازی شده برای عملکرد سریع

## 🛠️ تکنولوژی‌های استفاده شده

- [React](https://reactjs.org/) – کتابخانه‌ی جاوااسکریپت برای ساخت رابط کاربری
- [Tailwind CSS](https://tailwindcss.com/) – فریم‌ورک کاربردی CSS برای طراحی سریع
- [React Router](https://reactrouter.com/) – مدیریت مسیرها و ناوبری
- [Axios](https://axios-http.com/) (در صورت استفاده) – درخواست‌های HTTP
- [React Icons](https://react-icons.github.io/react-icons/) (در صورت استفاده) – آیکون‌های متنوع
- سایر کتابخانه‌ها: (لیست کتابخانه‌های استفاده شده را اضافه کنید)

## 📦 نصب و راه‌اندازی

برای اجرای پروژه روی کامپیوتر خود، مراحل زیر را دنبال کنید:

1. اطمینان حاصل کنید که [Node.js](https://nodejs.org/) و npm روی سیستم شما نصب است.
2. یک کپی از مخزن را clone کنید:
   ```bash
   git clone https://github.com/MobinDe/tour-torque.git
   
bash
cd tour-torque
وابستگی‌ها را نصب کنید:

bash
npm install
پروژه را در محیط توسعه اجرا کنید:

bash
npm start
با این کار پروژه روی http://localhost:3000 باز می‌شود.

📁 ساخت نسخه‌ی نهایی (Production)
برای ساختن فایل‌های بهینه‌شده جهت بارگذاری روی سرور، از دستور زیر استفاده کنید:

bash
npm run build
فایل‌های نهایی در پوشه‌ی build/ قرار می‌گیرند و می‌توانید آن‌ها را روی هر سرویس میزبانی استاتیک (مانند Vercel, Netlify) بارگذاری کنید.

🤝 مشارکت
پیشنهادات، درخواست‌ها و اصلاحات شما باعث بهبود پروژه می‌شود. لطفاً ابتدا یک issue ثبت کنید یا یک pull request ارسال نمایید.

📝 وضعیت پروژه
این پروژه در حال حاضر کامل شده و به‌عنوان نمونه‌کار (portfolio) ارائه می‌شود. توسعه‌ی آتی ممکن است شامل اتصال به backend واقعی، سیستم پرداخت و پنل مدیریت باشد.
